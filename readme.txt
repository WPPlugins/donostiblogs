Donostiblogs is a small plugin that shows a top-bar in your blog with links (icons) to other blogs in Donostia.

== Description ==

It is a plugin that shows the same floating (fixed) top bar in any blog with this plugin installed. This top-bar has links to another blogs or webpages. This links are updated remotely becouse it is designed for making a "connection" between diferent blogs in my city: San Sebastian-Donostia (Spain). The idea is to make a group of blogs with similar content or related content.

This bar is 22px height and it's situated in the top-most part of any page in the wordpress home. It doesn't need to change any code in the template/theme, only install and activate the plugin.

== Installation ==

1. Download the plugin / Descarga el plugin
2. Activate it / Activalo
3. It's done! / Ya est&aacute;!

**Manual Installation:**

1. Upload the `/donostiblogs/` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You don't have to do nothing, the bar shows when you activate the plugin.
